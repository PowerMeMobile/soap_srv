Messenger


Web service represents functions for sending SMS

The following operations are supported. For a formal definition, please review the Service Description.

  • Authenticate
    Authenticate the specified user

  • GetSmsStatus
    Returns status of specified transaction.

  • HTTP_Authenticate
    Authenticate the specified user

  • HTTP_GetSmsStatus
    Returns status of specified transaction.

  • HTTP_InboxProcessing
    Operate with Inbox

  • HTTP_KeepAlive
    Sends keep alive signal to server

  • HTTP_SendBinarySms
    Sends binary SMS using specified parameters.

  • HTTP_SendSms
    Sends SMS using specified parameters.

  • InboxProcessing
    Operate with Inbox

  • KeepAlive
    Sends keep alive signal to server

  • SendBinarySms
    Sends binary SMS using specified parameters.

  • SendServiceSms
    Sends service sms to recipient.

  • SendSms
    Sends SMS using specified parameters.

  • SendSms2
    Sends SMS using specified parameters.
